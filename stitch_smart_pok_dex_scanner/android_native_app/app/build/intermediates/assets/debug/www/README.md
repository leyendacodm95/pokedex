# Carpeta WWW (Assets)

Esta es la carpeta raíz donde Android buscará tu aplicación web.

## INSTRUCCIONES IMPORTANTES

1. Ve a la carpeta original de tu aplicación React (`...\stitch_smart_pok_dex_scanner\app\`).
2. Ejecuta `npm run build` en tu terminal.
3. Copia TODO el contenido de la carpeta `dist` generada (incluyendo `index.html` y la carpeta `assets`).
4. Pega todo el contenido directamente en esta carpeta (`android_native_app\app\src\main\assets\www\`).
5. Abre este proyecto en Android Studio y dale al botón de Compilar (o "Run")!
